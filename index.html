Developer / self-hosting setup

The UI must load the API from the same deployment as server.js (e.g. http://localhost:3000 via npm run dev). If you only uploaded HTML/JS/CSS to static hosting, /api/catalog will 404 — deploy the full Node app (Railway, Render, Fly.io, etc.) or set window.WORKOUTOS_API_BASE = "https://your-api.example.com" in this file before app.js when the API has another origin. First admin: npm run bootstrap-admin / ADMIN_EMAILS — see .env.example. Full deploy steps: DEPLOY.md in the repo.
